**Facebook Imitation**


Very Basic version of Facebook, has options such as adding to friends, chatting(implemented in SignalR), and more.


Technologies:


-ASP.NET CORE 5.23,


-ENTITY FRAMEWORK 6.13,


-SIGNALR 2.1.2,
